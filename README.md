# Unity Flappy Bird Clone

A simple Flappy Bird–style 2D game built using Unity and C#.

# Gameplay Features

Tap/press X to make the bird jump

Random pipe generation every 4 seconds

Pipes move continuously to the left

Collision detection → game restarts

Polished day background & ground sprite

# Core Scripts
✔ Bird.cs

Handles jump, input, collisions, score.

✔ Pipe.cs

Moves pipes left continuously.

✔ PipesGenerator.cs

Spawns new pipes using Coroutines with random heights.

# Technologies Used

Unity 2D

C#

Rigidbody2D Physics
